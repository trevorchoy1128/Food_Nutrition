## 格式要求

- **支援格式**：JPG、PNG、WebP
- **建議尺寸**：至少 200×200 像素（正方形裁切效果最佳）
- **檔案大小**：建議不超過 500KB（避免載入緩慢）

## 使用方法

1. 將照片改名為上表對應的檔案名稱
2. 放入此資料夾（`images/cases/`）
3. 重新載入 `reference.html` 即可自動顯示
4. 若不提供照片，系統會使用預設的網上圖片作備用
